# Python Learning Course - 2 Days

A comprehensive Python learning course designed to take you from beginner to intermediate level in just 2 days.

## 📚 Course Overview

This course is structured as a hands-on learning experience with Jupyter notebooks containing explanations, code examples, and exercises.

### Day 1: Python Basics (4 hours)

**File:** `day1_python_basics.ipynb`

Learn the fundamental concepts of Python programming:

- **Variables & Data Types** - Strings, integers, floats, booleans
- **Operators** - Arithmetic, comparison, logical operators
- **Control Flow** - If/else statements, loops (for, while)
- **Functions** - Creating reusable code blocks
- **Data Structures** - Lists, tuples, dictionaries, sets
- **String Manipulation** - Formatting, methods, operations
- **Error Handling** - Try/except blocks

### Day 2: Intermediate Python (4 hours)

**File:** `day2_intermediate_python.ipynb`

Build on your foundation with more advanced topics:

#### Part 1: Intermediate Concepts (60 minutes)

- **Advanced Data Structures** - Lists, tuples, sets operations
- **File Handling** - Reading, writing, appending, deleting files
- **Object-Oriented Programming (OOP)** - Classes, objects, inheritance

#### Part 2: Essential Libraries (180 minutes)

- **Pandas** - Data manipulation and analysis
  - Creating and inspecting DataFrames
  - Handling missing data
  - Selecting and filtering data
  - Working with CSV files
- **NumPy** - Numerical computing
  - Array creation and operations
  - 2D arrays (matrices)
  - Matrix multiplication
  - Mathematical functions
- **Matplotlib** - Data visualization
  - Line plots
  - Pie charts
  - Bar graphs (vertical, horizontal, grouped)
  - Customizing visualizations

## 🚀 Getting Started

### Prerequisites

- Python 3.7 or higher
- Jupyter Notebook or JupyterLab

### Installation

1. **Clone or download this repository**

2. **Create a virtual environment** (recommended):

```bash
python -m venv venv
```

3. **Activate the virtual environment**:

- Windows: `venv\Scripts\activate`
- Mac/Linux: `source venv/bin/activate`

4. **Install required packages**:

```bash
pip install -r requirements.txt
```

5. **Launch Jupyter Notebook**:

```bash
jupyter notebook
```

## 🔧 Troubleshooting

### Jupyter Notebook won't start

- Make sure you've activated your virtual environment
- Try: `pip install --upgrade jupyter`

### Import errors

- Ensure all packages are installed: `pip install -r requirements.txt`
- Check you're using the correct Python environment

### Kernel issues

- Restart the kernel: Kernel → Restart
- Clear output and restart: Kernel → Restart & Clear Output
